# Quest Tracker

Check off quests as you complete them. Pay attention to faction warnings!

## Main Story Quests

These advance the main plot. Some are points of no return.

- [ ] **Out of Time** - Leave Vault 111, go to Sanctuary
- [ ] **When Freedom Calls** - Help Preston in Concord
- [ ] **Jewel of the Commonwealth** - Go to Diamond City
- [ ] **Unlikely Valentine** - Find Nick Valentine in Vault 114
- [ ] **Getting a Clue** - Investigate Kellogg with Nick
- [ ] **Reunions** - Confront Kellogg at Fort Hagen
- [ ] **Dangerous Minds** - Use Memory Den to access Kellogg's memories
- [ ] **The Glowing Sea** - Find Virgil in the Glowing Sea
- [ ] **Hunter/Hunted** - Track down a Courser
- [ ] **The Molecular Level** - Build the Signal Interceptor (choose a faction to help!)
- [ ] **Institutionalized** - Enter the Institute, meet Father

**After Institutionalized, faction paths diverge. Choose wisely!**

## Point of No Return Warnings

**These quests will make factions permanently hostile:**

| Quest | Faction Lost |
|-------|--------------|
| Mass Fusion (for Institute) | Brotherhood of Steel |
| Spoils of War (for BoS) | Institute |
| Tactical Thinking | Railroad |
| End of the Line | Railroad |
| Precipice of War | Brotherhood of Steel |
| Rockets' Red Glare | Brotherhood of Steel |

**Safe until warned:** The game gives you a clear warning before any point of no return. Save your game when you see it!
