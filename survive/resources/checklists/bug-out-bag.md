# Bug-Out Bag Checklist

## Core Survival (The 5 Cs)
- [ ] Cutting tool (knife)
- [ ] Combustion device (fire starter)
- [ ] Cover (shelter/clothing)
- [ ] Container (water vessel)
- [ ] Cordage (550 cord/rope)

## Water
- [ ] Water bottles (2L minimum)
- [ ] Water filter
- [ ] Purification tablets
- [ ] Metal container for boiling

## Fire
- [ ] Lighter (2+)
- [ ] Ferro rod
- [ ] Waterproof matches
- [ ] Tinder (fatwood, dryer lint)

## Shelter
- [ ] Tarp/poncho
- [ ] Emergency blanket
- [ ] 550 paracord (100ft)
- [ ] Sleeping bag/bivy

## Food
- [ ] High-calorie bars
- [ ] MREs/freeze-dried meals
- [ ] Snares/fishing kit
- [ ] Mess kit

## First Aid
- [ ] Trauma kit
- [ ] Personal medications
- [ ] Antibiotics (if available)
- [ ] Pain relievers

## Navigation
- [ ] Map of area
- [ ] Compass
- [ ] GPS device
- [ ] Whistle

## Tools
- [ ] Multi-tool
- [ ] Duct tape
- [ ] Wire (snare/repair)
- [ ] Sewing kit

## Personal
- [ ] ID/documents (waterproof)
- [ ] Cash
- [ ] Emergency contacts
