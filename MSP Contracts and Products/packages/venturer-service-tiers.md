### Who Should Choose Premium?

**Premium is right for you if:**

- You need **24/7 critical support**—problems can't wait until morning

## Minimum Standards (All Tiers)

All managed devices and users must meet these standards:

### Hardware
- Computers less than 4 years old
- Business-grade equipment recommended

### Operating Systems
- Windows 11 Professional (Home upgraded during onboarding at client cost)
- macOS: Current version minus 1 (e.g., if current is Sequoia, Sonoma accepted)

### Security
- VSA X agent on all managed devices
- Datto EDR on all managed devices
- Microsoft 365 Business Basic/Standard minimum
- 1x Microsoft Entra ID P1 license per tenant (MFA, conditional access)
- No local administrator access (owner exception with documented liability acknowledgement)
- Password manager required (recommended: organisation-wide deployment)
- Licensed software only—no torrented or pirated software

### Travel
- VPN required for any overseas travel
