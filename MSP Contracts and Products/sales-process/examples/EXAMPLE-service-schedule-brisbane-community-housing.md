## 8. MINIMUM STANDARDS

The Client environment must meet these standards for full service delivery:

### 8.1 Hardware
- Computers less than 4 years old
- Business-grade equipment recommended

### 8.2 Operating Systems
- Windows 11 Professional (Windows Home to be upgraded at Client cost)
- macOS: Current version minus 1

### 8.3 Microsoft 365
- Microsoft 365 Business Basic or higher
- One (1) Microsoft Entra ID P1 license per tenant (for MFA, conditional access)

### 8.4 Security
- No local administrator access (owner exception with documented liability acknowledgement)
- Password manager required
- Licensed software only
- VPN required for overseas travel

### 8.5 Remediation
Any gaps identified during onboarding will be documented and remediated as part of transition or quoted as project work.
