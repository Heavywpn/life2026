# HR Issues Tracker

A simple, local HR tracking system compliant with the Australian Fair Work Act 2009 and Telecommunications Award MA000041.

## Getting Started

1. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari)
2. No installation or internet connection required
3. All data is stored locally in your browser

## Features

### Staff Information Questionnaire
- Personal details (name, contact, emergency contact)
- Employment details compliant with MA000041 classifications
- Superannuation and tax information
- Leave entitlements tracking (NES compliant)
- Work rights verification
- Compliance checklist (FWIS, contract, induction)

### HR Issue Recording
- Multiple issue categories (performance, conduct, workplace relations, compliance)
- Severity levels for prioritisation
- Fair Work compliance checklist:
  - Employee notification tracking
  - Natural justice (opportunity to respond)
  - Support person offered
  - Documentation provided
- Action tracking and follow-up dates

### Dashboard
- Overview of staff count and issue statistics
- Recent activity log
- Quick access to all sections

### Data Management
- Export all data to JSON file (for backup)
- Import data from JSON file
- All data stored locally (Privacy Act compliant)

## Compliance Features

### Fair Work Act 2009
- Natural justice requirements built into issue recording
- Support person reminder for meetings
- Unfair dismissal protection timeline tracking
- NES leave entitlements

### Telecommunications Award MA000041
- Classification levels (TO1-5, TT1-4)
- Minimum wage checking against approximate rates
- Ordinary hours limits (38 hrs/week max)
- Employment type tracking

### Record Keeping
- All records timestamped
- 7-year retention reminder on deletion
- Export function for archiving

## Important Notes

1. **This is a tool, not legal advice** - For complex HR matters, consult a qualified HR professional or employment lawyer

2. **Verify minimum wages** - The built-in rates are approximate. Always verify current rates at https://calculate.fairwork.gov.au

3. **Keep Fair Work Information Statement on hand** - Must be provided to all new employees

4. **Regular backups** - Use the Export function to backup your data regularly

5. **Privacy** - Data is stored only in your browser's local storage. Clear browser data will delete records.
