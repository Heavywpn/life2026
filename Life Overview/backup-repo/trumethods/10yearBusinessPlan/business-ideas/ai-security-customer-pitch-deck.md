## Presentation Tips

### For In-Person Meetings:

**Setup:**
- Laptop with slides (PDF backup if needed)
- Business cards ready
- Information packet printed
- Notebook for taking notes
- Phone on silent

**Body Language:**
- Sit beside them, not across (collaborative, not adversarial)
- Show slides on their screen if possible (their territory, comfortable)
- Pause for questions (make it conversation, not lecture)
- Watch for confusion (adjust language if too technical)

**Pacing:**
- Don't rush (they need to absorb information)
- Check understanding ("Does this make sense so far?")
- Adapt to interest level (skip ahead if they're engaged, slow down if confused)

### For Workshops (Group Presentations):

**Engagement:**
- Start with question: "Who here is using AI in their business?" (hands up)
- Show live demo (actually try to hack a chatbot - exciting!)
- Encourage questions throughout (not just at end)
- Use Brisbane-specific examples (local context matters)

**Ending:**
- Clear call to action (free consultation signup sheet)
- Make it easy (QR code, simple URL)
- Offer to stay after (one-on-one questions)
