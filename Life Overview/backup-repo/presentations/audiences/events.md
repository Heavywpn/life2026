## Session Structure Templates

### 20-Minute Conference Session
**0-3 min**: Hook and credibility
- Powerful opening
- Military background reveal
- Why this matters today

**3-12 min**: Core content
- 3 key insights
- Real examples
- Actionable takeaways

**12-18 min**: Deep dive or case study
- Specific application
- Results/outcomes
- Lessons learned

**18-20 min**: Close and Q&A setup
- Summary
- Call to action
- Open for questions

### 45-Minute Deep Dive
**0-5 min**: Opening and context
**5-15 min**: Problem/challenge overview
**15-35 min**: Solution framework (with examples)
**35-40 min**: Implementation guidance
**40-45 min**: Q&A

### 5-Minute Booth Presentation
**0-1 min**: Attention grabber
**1-3 min**: Problem + unique solution
**3-4 min**: Quick demo or example
**4-5 min**: Clear next step

## Presentation Design for Events

### Visual Style
- Professional but bold
- High-impact imagery
- Minimal text on slides
- Strong color contrast
- Military/tech visual balance

### Key Slides
1. Title with your photo (both military and current)
2. The "expectation vs reality" reveal
3. Key statistics or facts
4. Case study visuals
5. Framework/process diagrams
6. Takeaway summary
7. Contact/follow-up

### Technical Considerations
- Have backup of presentation (USB + cloud)
- Work without internet if needed
- Test on venue equipment if possible
- Bring own clicker/remote
- Have handout version ready

## Common Event Scenarios

### You're Running Over Time
- Jump to conclusion
- Skip lower-priority content
- Speed up without rushing
- Respect the schedule

### Technical Failure
- Have backup plan
- Don't panic
- Can you present without slides?
- Use it as opportunity to show adaptability
- "In the military, plans never survive contact..."

### Tough Questions
- Acknowledge the question
- Be honest if you don't know
- Offer to follow up
- Turn to your experience
- Don't defensive

### Low Attendance
- Don't let it show disappointment
- Make it intimate/valuable
- More time for questions
- Better networking opportunity
- Quality over quantity
