## Appendix B: Minimum Standards

To ensure we can deliver quality service, all managed environments must meet these standards:

### Hardware
- Computers less than 4 years old
- Business-grade equipment recommended

### Software
- Windows 11 Professional (Home upgraded during onboarding at client cost)
- macOS: Current version minus 1
- Microsoft 365 Business Basic or higher
- One Entra ID P1 license per tenant (for MFA, conditional access)

### Security
- No local administrator access (owner exception documented)
- Password manager required
- Licensed software only

*Any gaps identified during onboarding will be addressed as part of transition.*
