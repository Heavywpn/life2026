# MSP Contracts and Products

This project contains the contracts, service packages, and pricing frameworks for Venturer Technology's managed services offerings in Australia.

---

## Project Purpose

This project supports Venturer Technology's strategic transformation from a commodity MSP ($45/seat) to Queensland's most trusted MSP partner for charities and mission-driven organizations ($130-150/seat premium pricing).

**Key Objectives:**
1. Create Australian-compliant MSP contracts (ACL, Privacy Act, NDB scheme)
2. Define service packages aligned with TruMethods "Four Blocks" framework
3. Establish clear "edges" for service delivery
4. Support the 10-year business plan transformation journey

## Australian Legal Compliance

All contracts in this project are designed to comply with:

- Australian Consumer Law (ACL) - unfair contract terms
- Privacy Act 1988 (Cth) - Australian Privacy Principles
- Notifiable Data Breaches (NDB) scheme
- Electronic Transactions Act 1999 (Cth)
- ACSC Essential Eight security guidelines
- Industry-specific regulations (healthcare, finance, government)

**Important:** Final contracts should be reviewed by a qualified Australian legal practitioner before execution.

## Service Framework: The Four Blocks

Based on TruMethods methodology, all packages deliver:

| Block | Description |
|-------|-------------|
| **Centralized Services** | RMM, monitoring, patching, backup, security tools |
| **Technology Alignment** | Proactive management, best practices, standardization |
| **Support** | Helpdesk, remote/onsite service, issue resolution |
| **Virtual CIO** | Strategic planning, technology roadmap, business alignment |

**Key Differentiator:** Professional and Premium tiers include vCIO services bundled in. Essential tier includes quarterly check-ins and annual technology reviews to maintain partnership relationship.

---

## Quick Reference: Service Tiers

| Tier | Target | Monthly AISP | Key Features |
|------|--------|--------------|--------------|
| **Essential** | 1-3 staff | $85/user | Business hours support, Datto EDR, quarterly check-ins |
| **Professional** | 4-50 staff OR high-compliance | $130/user | 24/7 SOC, full security stack, vCIO, QBRs, Essential 8 |
| **Premium** | 50+ staff | $175-220/user | Dedicated TAM, 24/7 critical, advanced compliance |

**Note:** Financial services, Legal, NDIS, Government-funded organisations must use Professional or higher.

## Usage

1. **New Client:** Use discovery questionnaire → select appropriate tier → generate proposal → customize service schedule
2. **Existing Client Conversion:** Assess current state → map to new tier → create transition plan → update agreements
3. **Price Increase:** Follow annual adjustment process with appropriate notice periods
