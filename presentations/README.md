# Marketing Presentations Project

## Purpose
Central hub for building, tracking, and refining presentation ideas for marketing Venturer Technology.

## Project Structure

```
presentations/
├── README.md                          # This file - project overview
├── tracker.md                         # Status tracking for all presentations
├── templates/                         # Reusable presentation templates
├── active/                           # Currently being developed
│   └── bni-presentation-ideas.md    # BNI Friday presentation
├── archive/                          # Completed presentations
├── resources/                        # Supporting materials
│   ├── key-messages.md              # Core company messages
│   ├── visual-assets.md             # Image/video inventory
│   └── story-bank.md                # Personal stories & anecdotes
└── audiences/                        # Audience-specific guidelines
    ├── bni.md                       # BNI chapter presentations
    ├── prospects.md                 # Client prospect meetings
    └── events.md                    # Industry events & conferences
```

## Quick Start

### Creating a New Presentation
1. Copy template from `templates/presentation-template.md`
2. Place in `active/` folder
3. Add entry to `tracker.md`
4. Develop and refine
5. Move to `archive/` when delivered

### Key Principles
- Lead with authenticity (military background, fairness focus)
- Connect personal story to business value
- Make it memorable, not generic
- Always have clear call-to-action

## Core Narrative Elements
- 10 years Australian Infantry (belt-fed machine gun line)
- East Timor liberation experience
- Fairness as driving principle
- Business partnership approach
- IT as removing barriers, not fixing people
