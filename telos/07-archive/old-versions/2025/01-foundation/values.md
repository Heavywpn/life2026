## Decision Framework

*Using values to make tough choices*

When facing a difficult decision:

1. Which values does option A honor?
2. Which values does option B honor?
3. Which values would I violate with each option?
4. Which path aligns with my highest-priority values?
5. Can I find a third option that honors more values?
